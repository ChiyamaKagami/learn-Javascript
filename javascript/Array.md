# Array

## 方法
- `copyWithin`
在数组内部，将一段元素序列拷贝到另一段元素序列上，覆盖原有的值。
-  `fill`
将数组中指定区间所有元素的值，都替换成某个固定的值。
- `pop`
- `push`
- `reverse`
- `shift`
- `sort`
- `splice`
- `unshift`

## 访问方法
- `concat`
用于合并两个或多个数组。此方法不回更改现有数组，而是返回一个新数组。
- `includes`

- `join`

- `slice`

- `toString`
由所有数组元素组合而成的字符串。遮蔽原型链上的`toString`方法
