# Module
模块化就是将一个复杂的程序依据一定的规则封装成几个块(文件)，并将其组合在一起。  
块内部数据和实现是私有的，是指向外暴露一些接口(方法)以此来与其他模块通信。
## COMMON
common.js的历史最早，解决js的模块化问题。
但是有同步问题，所以在浏览器一般不使用。
```
var math = require('math');
math.add(2,3);
```
## AMD
遵循RequireJS规范，推崇依赖前置(提前执行)  
`优点`:  
+ 实现js的异步加载，避免网页失去响应
+ 管理模块之间的依赖性，按照依赖关系加载，便于代码的编写和维护。
## CMD
遵循SeaJS规范，推崇依赖就近(延迟执行)
(按需加载)
