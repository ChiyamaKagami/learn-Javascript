# 胶卷内容提取
## 粗略步骤
1. 反相(负片)
2. 图像矫正
3. 内容获取

## 具体步骤(轮廓)
1. 图像反相
2. 灰度化
3. 阈值二值化
4. 检测轮廓
5. 寻找轮廓包围矩阵，获取角度
6. 根据角度旋转矫正
7. 截取图像区域

## 具体步骤(霍夫变换)
1. 霍夫线变换探测直线
2. 计算直线倾斜角，获取平均值
3. 根据倾斜角旋转矫正
4. 裁剪内容

### 图像矫正

### 内容获取
#### 特征分析
- 颜色特征(片基基本都是黑色或者偏黑色)
- 大小特征(135相纸的长宽固定)
- 形状特征(矩形)
- ROI感兴趣区域
- 轮廓检测
- Canny算法
- findContours 找联通区域
- approxPolyDP来拟合多边形

- 识别白边
- 判断白边长度/阈值
