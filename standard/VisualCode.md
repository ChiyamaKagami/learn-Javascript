# Hot Key
ctrl + U 光标撤销
ctrl + D 将下一个查找匹配项添加到选择
ctrl + KD查找下个匹配项
ctrl + KL切换折叠方式
ctrl + PageUp/PageDown 切换编辑器
ctrl + B 切换侧边栏可见性
shift + F6 / F6 切换视图专注
ctrl + shift + Enter 向上插入行
ctrl + Enter 向下插入行
# Emmet
提高代码编写速度的工具。
## 基本语法
|value|意义|
-|-|
|`E`|HTML标签|
|`E#id`|id属性|
|`E.class`|class属性|
|`E[attr=foo]`|特定属性|
|`E{foo}`|标签包含内容是foo|
|`E>N`|N是E的子元素|
|`E+N`|N是E的同级元素|
|`E^N`|N是E的父元素|
