# Required
## manifest_version
文件版本，Chrome 18后设置为2(不带引号)。
## name
插件名称
## version
插件版本
# Recommended
## default_locale
指定语言环境
- en
- zh_CN
## description
插件描述(最大132字符)
## icons
图标
```
"icons": {
  "16": "icon16.png",
  "48": "icon48.png",
  "128": "icon128.png"
}
```
## Pick one (or none)
## browser_action
作为工具栏的插件，除图标之外，浏览器操作还能是工具栏提示、徽章、弹窗
