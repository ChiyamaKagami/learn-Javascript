# 图表批量导出
- Customizer
负责获取数据。
- Adapter
处理获取到的数据，生成渲染任务。
- Dominator
负责整个渲染过程的调配。
- Painter
负责绘制。
# 流程
1. Dominator接收到绘制任务
2. Customizer获取相应的数据并处理成统一格式
3. Adapter接受格式化好的数据，并将数据转换成渲染的数据
4. Painter接收到渲染数据，将数据置入需要渲染的图表类型模版中，并开始渲染
